The second derivative test uses f"(x) to find relative extrema.

Procedure:
  1. Find critical values of f'(x)
  2. Find f"(x)
  3. Plug in critical values to f"(x)
Results:
  • If f"(x) < 0 then the critical value is a relative max.
  • If f"(x) > 0 then the critical value is a relative min.
  • If f"(x) = 0 then the test is inconclusive.
Note: Critical values are not guaranteed extrema, but all relative extrema will be at critical values.
