Steps In Analyzing a Graph:
  1. Slope: Find f'(x)
  2. Find values that make f'(x) = 0 or undefined (critical values)
  3. Establish intervals using critical values
  4. Test intervals

Concavity:
  • Whether the graph is hugging up or down.
  • Uses the second derivative, f"(x). Follow the same process as for f'(x)
