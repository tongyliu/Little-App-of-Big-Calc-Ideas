"How many times a particular number is a zero for a given polynomial."

For Example:

Consider y=(x-r)^n
If n=1, then r is a simple root. (Passes through x-axis only once)
If n is odd and > 1, then the graph passes through x-axis at r and has an inflection point there.
If n is even, then the graph is tangent to the x-axis at r.
